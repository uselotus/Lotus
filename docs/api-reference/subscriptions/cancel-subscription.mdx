---
title: "Cancel a subscription"
description: "Cancels a subscription. You can modify the behavior of the cancellation based on
the arguments passed to the call."
openapi: "POST /api/subscriptions/{subscription_id}/cancel/"
---

<CodeGroup>

```py Python
lotus.cancel_subscription(
    subscription_id='sub_a623349004cd4947aca1851c64aa6fbd',
    bill_usage=True,
    invoicing_behavior='add_to_next_invoice',
)
```

```ts Typescript

await lotus.cancelSubscription({
  subscription_id: "plan234234"
  bill_usage: "false";
  invoicingBehavior?: "add_to_next_invoice";
  flatFeeBehavior:"prorate";
  usageBehavior:"bill_none"
});
```

</CodeGroup>
