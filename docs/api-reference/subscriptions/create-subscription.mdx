---
openapi: "POST /api/subscriptions/"
title: Create a subscription
description: Add a plan to a customer's subscription
---

A subscription associates one of your customers with one of your billing plans. You can define multiple subscriptions of the same plan for a ssingle customer by using the subscription filters.

<CodeGroup>
```py Python
lotus.create_subscription(
  customer_id='cust_6c237d6c149c47db8f64246b78ecac13',
  plan_id='plan_f17c6153807d4f4b812265a09cf06680',
  start_date=datetime.datetime(2022, 2, 23, 0, 0),
)
```

```ts Typescript
await lotus.createSubscription({
  customerId: "customer_123",
  planId: "billing_plan_5",
  startDate: "2022-02-23",
});
```

</CodeGroup>
