---
title: "Update a subscription"
openapi: "POST /api/subscriptions/{subscription_id}/update/"
description: "Cancel auto-renew or change the end date of a subscription."
---

<CodeGroup>

```py Python
lotus.update_subscription(
  subscription_id='sub_a623349004cd4947aca1851c64aa6fbd',
  turn_off_auto_renew=True
)

lotus.update_subscription(
  subscription_id='sub_a623349004cd4947aca1851c64aa6fbd',
  end_date=datetime.date(2022, 12, 8),
)
```

```ts Typescript
await lotus.updateSubscription({
  subscription_id: "sub_a623349004cd4947aca1851c64aa6fbd";
  turn_off_auto_renew: true;
});
```
